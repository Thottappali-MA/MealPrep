# Meal Prep App

A warm, supportive meal-planning app designed for busy professionals.

## Features

- **Weekly Meal Planning**: AI-assisted planning (Mocked for MVP).
- **Recipe Library**: Simple and healthy recipes.
- **Grocery List**: Auto-generated from your plan.
- **Home Dashboard**: Quick access to today's meal and tasks.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run android
   # or
   npm run ios
   # or
   npm run web
   ```

## Project Structure

- `src/screens`: Application screens (Home, Planner, Recipes, Grocery).
- `src/navigation`: Navigation configuration.
- `src/components`: Reusable components.
- `src/constants`: Mock data and constants.
- `src/types`: TypeScript definitions.

## Tech Stack

- React Native (Expo)
- TypeScript
- React Navigation
- React Native Paper (UI Library)
